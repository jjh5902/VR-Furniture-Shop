# VR-Furniture-Shop

## VR을 이용한 가구 쇼핑
- VR을 통해 입체적으로 가구를 살펴보고 집에서도 주문 가능하게 할 수 있음
- 가구 재비치를 통해 좀 더 생생한 가구 쇼핑을 할 수 있음
